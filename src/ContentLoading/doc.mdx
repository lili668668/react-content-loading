# ContentLoading

## Description

A loading component.

## Props
<PropsTable of={ContentLoading} />

## Basic usage

`import ContentLoading from 'react-content-loading/lib/ContentLoading'` or `import { ContentLoading } from 'react-content-loading'`

<Playground>
  <ContentLoading />
</Playground>

## Advanced usage

### Change background color

<Playground>
  <ContentLoading placeholderColor="#fecdcd" />
</Playground>

### Change loading color

<Playground>
  <ContentLoading placeholderColor="#fecdcd" loadingColor="#ff9393" />
</Playground>

### Change the height and width

<Playground>
  <ContentLoading
    placeholderColor="#fecdcd"
    loadingColor="#ff9393"
    height="20px"
    width="400px"
  />
</Playground>
